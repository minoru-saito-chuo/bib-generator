# BibGenerator

BibGeneratorは、LaTeXの参考文献リスト（`thebibliography` 環境内の `\bibitem`）をGUIで簡単に作成・管理・エクスポートできるWebアプリケーションです。

手書きによるフォーマットミスを防ぎ、論文執筆時の参考文献管理を効率化します。

## 🚀 利用はこちら

**[BibGenerator を開く](https://minoru-saito-chuo.github.io/bib-generator)**
(GitHub Pages)

## ✨ 主な機能

* **直感的な編集UI**: フォームに入力するだけで、正しいフォーマットの `\bibitem` コードをリアルタイム生成。
* **論文 / Web サイト対応**: 文献の種類（Paper/Web）に応じた入力フィールドの切り替え。
* **言語サポート**: 日本語（`ujarticle`等向け）と英語（標準的なTeXスタイル）のフォーマット切り替え。
* **ドラッグ＆ドロップ**: 参考文献の順序を直感的に並び替え可能。
* **インポート / エクスポート**:
    * 生成された `.tex` ファイルのダウンロード（再編集用データの埋め込み対応）。
    * 既存データのインポート（ファイルアップロードまたはテキスト貼り付け）。
    * クリップボードへのコピー。
* **一時保存**: Cookieを使用したブラウザへのデータ一時保存機能。
